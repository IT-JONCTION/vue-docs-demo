    MyStack is a flexible container component that allows easy styling of child elements.

    Props

    Name Type Default value Required? Description
    spacing Number 2 No Specifies the spacing between child elements. Multiplier of 0.25rem.
    wrap Boolean false No Determines whether child items should wrap to the next line/column.
    direction String "row" No Direction of the child elements, can be either "row" or "column".
